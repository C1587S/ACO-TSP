Instalación¶

Paquete de Python¶

Instala el pquete con:

#ejecuta la sigueinte línea desde el directorio `src` donde está `setup.py`
#luego de clonar el repo:
#https://github.com/optimizacion-2-2021-1-gh-classroom/practica-1-segunda-parte-ltejadal.git
pip install .

o con el editable:

#ejecuta la sigueinte línea desde el directorio `src` donde está `setup.py`
#luego de clonar el repo:
#https://github.com/optimizacion-2-2021-1-gh-classroom/practica-1-segunda-parte-ltejadal.git
pip install -e .

o con la url:

pip install "git+https://github.com/optimizacion-2-2021-1-gh-classroom/practica-1-segunda-parte-ltejadal.git#egg=opt2&subdirectory=src"

"ant_colony"
